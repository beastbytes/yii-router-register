# Yii RouterRegister
Yii RouterRegister allows generation of Yii router configuration
using PHP [Attributes](https://www.php.net/manual/en/language.attributes.php)
amd [Enumerations](https://www.php.net/manual/en/language.enumerations.php).

It provides an implementation of RouterRegister/WriterInterface to write Yii Router configuration files,
and a console command to run RouterRegister.

## Installation
Install RouterRegister using [Composer](https://getcomposer.org/)

Add the following to the require-dev section of your composer.json:

```json
"beastbytes/yii-router-register": "<version-constraint>",
```

or run

```PHP
php composer.phar require -dev "beastbytes/yii-router-register:<version-constraint>"
```
